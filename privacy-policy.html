Privacy Policy for YouTube Q&A Assistant

Last Updated: November 9, 2025

Introduction

YouTube Q&A Assistant ("we", "our", or "the extension") is a Chrome browser extension that provides AI-powered question and answer functionality for YouTube videos. This privacy policy explains how we handle your data.

Data Collection

The extension collects and processes the following data:

  • Video Transcripts: Extracted client-side from YouTube videos you visit
  • Questions: Questions you ask about videos
  • Video IDs: YouTube video IDs for conversation context
  • Backend API URL: Your configured backend API URL (stored locally)

How Data is Used

  • Video transcripts are extracted in your browser (client-side) from YouTube
  • Transcripts and questions are sent to your configured backend API server
  • The backend processes data using Azure OpenAI to generate answers
  • Conversation history is stored on your configured backend server for context
  • No data is stored locally by the extension itself

Data Storage

  • Local Storage: Only your configured backend API URL is stored locally using Chrome's storage API
  • Backend Server: Conversation history and transcripts are stored on your configured backend server
  • No Third-Party Storage: We do not store your data on any third-party servers

Third-Party Services

  • YouTube: For video transcript extraction (client-side)
  • Backend API: Your configured backend server (default: Azure VM)
  • Azure OpenAI: Used by backend for AI processing (if using default backend)

User Control

  • You can configure your own backend URL in extension settings
  • You can clear conversation history at any time
  • Extension only works on YouTube video pages you actively visit
  • You can uninstall the extension at any time

Data Security

  • Transcripts are extracted client-side in your browser
  • All API communications use HTTPS encryption
  • You control which backend server receives your data

Children's Privacy

This extension is not intended for users under 13 years of age. We do not knowingly collect data from children.

Changes to Privacy Policy

We may update this privacy policy. Changes will be posted on this page with an updated "Last Updated" date.

Compliance

This extension complies with:

  • Chrome Web Store Developer Program Policies
  • General Data Protection Regulation (GDPR) principles
  • California Consumer Privacy Act (CCPA) principles

Contact

For questions about this privacy policy, please contact us through the extension's support channel or GitHub repository.
